# 波のシミュレーション
* 物理基礎の波動分野の理解を深めるための教材です。
* 各点の媒質は、その場で振動していることを確認してください。
* 初めて物理基礎を学ぶ高校生にとっては、パルス波モードを使うと、$y-x$グラフや$y-t$グラフのイメージがしやすいかもしれません。
  * 画像表示のチェックボックスを使って、視点を切り替えましょう。
